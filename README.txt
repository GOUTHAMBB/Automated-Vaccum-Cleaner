<h1 style="font-size:24px;">Automatic Vacuum Cleaner</h1>

<h2 style="font-size:20px;">Overview</h2>
<p>The Automatic Vacuum Cleaner is a project aimed at automating the process of cleaning floors using robotics and sensors. This project utilizes an Arduino Uno microcontroller along with various electronic components to create a fully automated vacuum cleaner.</p>

<h2 style="font-size:20px;">Features</h2>
<ul>
  <li><strong>Start-up Sequence:</strong> The vacuum cleaner initializes upon power-on and performs self-checks.</li>
  <li><strong>Battery Status Check:</strong> It checks the battery status before starting the cleaning process.</li>
  <li><strong>Cleaning Functionality:</strong> Upon confirming sufficient power, the vacuum cleaner starts cleaning by navigating the environment.</li>
  <li><strong>Obstacle Avoidance:</strong> Ultrasonic sensors assist the vacuum cleaner in detecting and avoiding obstacles in its path.</li>
  <li><strong>High-speed Suction:</strong> A high-speed DC motor creates suction to collect dust and debris.</li>
</ul>

<h2 style="font-size:20px;">Components</h2>
<ul>
  <li>Arduino Uno</li>
  <li>Four servo motors</li>
  <li>H-Bridge</li>
  <li>Three ultrasonic sensors</li>
  <li>High-speed DC motor</li>
</ul>

<h2 style="font-size:20px;">Functionality</h2>
<p>The Arduino Uno serves as the brain of the vacuum cleaner, coordinating the operation of all components. The servo motors control movement, while the H-Bridge regulates the DC motor's speed for suction. Ultrasonic sensors provide input for obstacle detection, enabling the vacuum cleaner to navigate autonomously.</p>

<h2 style="font-size:20px;">Usage</h2>
<p>To use the Automatic Vacuum Cleaner:
<ol>
  <li>Connect the necessary components as per the circuit diagram.</li>
  <li>Upload the Arduino sketch to the Arduino Uno.</li>
  <li>Power on the vacuum cleaner.</li>
  <li>Follow the instructions for initiating the cleaning process.</li>
</ol>
</p>

<h2 style="font-size:20px;">Installation</h2>
<p>Ensure you have the Arduino IDE installed on your computer. Download the required libraries for the project and include them in your Arduino sketch.</p>
